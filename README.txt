How to input the source code into Eclipse:
 1. un-zip the folder into a suitable location
 2. Create a new empty Java project into the Eclipse workspace (File->New->Java Project).
 3. Right-click on your Java project in Eclipse and select Import->General->File system,
	then choose the 'export' folder and click 'Finish'.
 4. Run the application from the class: 'Main.java'

Note: The JUnit tests will not work
 

How to run IslandTrader.jar:
 1. Ensure Java 14 is installed on the machine.
 2. Open the zip folder in a terminal window.
 3. Run "java -jar IslandTrader.jar" for a GUI, or "java -jar IslandTrader.jar cmd" for command line user interface.
